# Remote Demo Video Shot List

Use this shot list for a remote demo video that does not require conference attendance.

## Required Captures

1. README section describing CASR.
2. Dashboard with "Create CASR Demo".
3. Demo project creation confirmation.
4. Episode workbench with CASR panel.
5. CASR analysis action.
6. Global quality score and continuity score.
7. Shot-level failure attribution.
8. Strategy tree with recommended repair path.
9. Cost, time, risk, success probability, and expected gain.
10. User-confirmed execution or safe simulated execution result.

## Recording Rules

- Do not show API keys, JWT tokens, database passwords, or local secrets.
- Use a clean demo account.
- Keep the browser in 16:9 layout.
- Keep the final video under 5 minutes.
- Upload to a stable link that can be used in README and arXiv ancillary notes.
