# Wan2.2 数据说明

这里提交的是模板，不提交真实视频、图片、权重和训练输出。

## 数据拆分

按项目拆分：

- `train`: 80%
- `val`: 10%
- `test`: 10%

不要让同一项目或同一角色同时出现在训练集和测试集里。

## 可用于 LoRA 的样本

`usable_for_lora=true` 必须同时满足：

- `license_status` 是 `owned` 或 `licensed`。
- `quality_label=ok`。
- 竖屏 9:16。
- 5-8 秒单镜头。
- 无水印、无字幕遮脸。
- 人物/服装/场景稳定。

失败样本只进入质量门禁，不进入 LoRA 正样本。
